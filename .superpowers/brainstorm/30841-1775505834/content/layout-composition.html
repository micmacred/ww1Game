A: Map-Dominant + Side Panel

Map takes ~75% width. Narrow right panel for turn info, sector details, and actions. Tapping a sector populates the side panel. Resource allocation opens as an overlay.

Maximises the war-room table feel. Side panel risks feeling cramped for allocation controls.

BEGIN TURN

B: Full-Width Map + Bottom Drawer

Map spans entire width at ~65% height. Bottom third is a persistent drawer for turn state, selected sector, and actions. Allocation expands the drawer upward over the map.

Map reads better at full width. Bottom drawer mirrors how the campaign layer's reserves panel works.

TURN 3

C: Full-Screen Map + Phase Overlays

Map fills entire screen. Turn info as a small badge. Tapping a sector opens a detail popover. Allocation phase slides in as a translucent overlay. Results phase overlays the map with animated updates.

Most immersive war-room feel. Risk: overlays can feel like interruptions rather than part of the flow.

Key trade-off

More map = more immersive, but UI controls need to live somewhere. The campaign layer learned that burying controls leads to confusion. The question is whether theatre's slower pace (no timer pressure) allows overlays where campaign couldn't.
